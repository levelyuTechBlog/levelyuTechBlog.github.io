在linux下搭建Git服务器

git服务器环境

服务器 CentOS7 + git（version 1.8.3.1）
客户端 Windows10 + git（version 2.16.0.windows.2）

安装Git

Linux 做为服务器端系统，Windows 作为客户端系统，分别安装 Git
服务器端：

# yum install -y git

安装结束后，查看Git版本

# git --version
git version 1.8.3.1

客户端：
下载 Git for Windows，地址：下载地址
安装完之后，可以使用 Git Bash 作为命令行客户端。
安装完之后，查看 Git 版本

$ git --version
git version 2.16.0.windows.2
